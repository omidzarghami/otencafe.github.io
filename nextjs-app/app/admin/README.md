# پنل مدیریت کافه اوتن

## دسترسی

- **URL**: `/admin/login`
- **رمز عبور پیش‌فرض**: `admin123`

⚠️ **هشدار امنیتی**: در production حتماً رمز عبور را تغییر دهید!

## ویژگی‌ها

### 1. صفحه ورود (`/admin/login`)
- ورود با رمز عبور
- Session management با cookies
- مدت اعتبار: 7 روز

### 2. داشبورد مدیریت (`/admin/dashboard`)
- مشاهده تمام آیتم‌های منو
- ویرایش قیمت‌ها
- ذخیره تغییرات
- خروج از سیستم

## نحوه استفاده

1. به آدرس `/admin/login` بروید
2. رمز عبور را وارد کنید
3. پس از ورود، به داشبورد هدایت می‌شوید
4. قیمت‌ها را ویرایش کنید
5. روی دکمه "ذخیره تغییرات" کلیک کنید

## تغییر رمز عبور

برای تغییر رمز عبور، فایل `app/api/auth/login/route.ts` را ویرایش کنید:

```typescript
// خط 15: رمز عبور را تغییر دهید
const isValid = password === "رمز-جدید-شما";
```

## ذخیره‌سازی داده‌ها

قیمت‌های به‌روزرسانی شده در فایل `data/prices.json` ذخیره می‌شوند.

⚠️ این فایل در `.gitignore` قرار دارد و commit نمی‌شود.

## امنیت

- تمام routes مربوط به admin با middleware محافظت می‌شوند
- Session با httpOnly cookies ذخیره می‌شود
- در production، از HTTPS استفاده کنید
